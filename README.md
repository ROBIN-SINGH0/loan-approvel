# Loan Approval System

## About
This project predicts **loan approval** using machine learning.  
It checks customer details and gives a **Yes / No** decision for loan approval.

## Tech Used
- Python  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

## Dataset
- Loan customer data (CSV file)

## Steps
- Load data  
- Clean data  
- Encode text data  
- Split data into train and test  
- Train ML model  
- Check accuracy  

## How to Run
1. Open the `.ipynb` file  
2. Keep the dataset in the same folder  
3. Run all cells  

## Result
The model predicts whether a loan is **Approved** or **Not Approved**.
